
## 项目介绍
    `pkg` 是海量数据高并发场景，构建Go+ES8企业级搜索微服务 项目基于 Golang开发的第三方sdk，自定义封装常用的组件代码

## 已实现包组件封装
    1. mysql
    2. redis
    3. elasticsearch
    4. kafka
    5. httpclient
    6. mongoDB
    7. routine 协程池
    8. shutdown 优雅关闭
    9. logger 日志库 
   等等

## 全面的敏捷开发特性

该项目提供了golang后端开发的全面支持，这些支持包括：
* 完善的后端组件支持
* 使用简单方便
* 能快速开发golang后端服务
方便在golang 项目中对 相关组件的重复利用
